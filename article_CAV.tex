\begin{abstract}

In this paper, we propose a new tool for the model-checking of
Process Hitting (PH), a recently introduced framework to model concurrent processes,
by using the logic programming paradigm called Answer Set Programming (ASP).
PH is a restriction of synchronous automata which is notably suitable,
but not limited, to model biological regulatory networks.
We focus on the problem of reachability that consists of deciding if,
starting from a given initial state, it is possible to reach a given (partial) state
in a given number of steps;
this is thus equivalent to the CTL “$\mathsf{EF}$” operator
with paths limited to a certain length.
We also consider the enumeration of fixed points, that is,
states in which no more evolution is possible.
The merits of our methods is illustrated by applying them to
biological examples of various sizes
and comparing the results with some existing approaches.
It turns out that our approach succeeds in processing large models
with a high number of components and interactions.

\keywords{Process Hitting, Answer Set Programming, stable states, fixed points, reachability}

\end{abstract}
